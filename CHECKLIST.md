## üÜò TROUBLESHOOTING

### **Problem: Website Doesn't Load**
- [ ] Wait 3-5 minutes after deployment
- [ ] Clear browser cache (Ctrl+Shift+Delete)
- [ ] Try incognito/private window
- [ ] Check repository is PUBLIC (not private)

### **Problem: Changes Don't Show**
- [ ] Wait 2 minutes for GitHub to rebuild
- [ ] Clear browser cache
- [ ] Check you committed changes
- [ ] Try different browser

### **Problem: Images Don't Show**
- [ ] Check images are uploaded to GitHub
- [ ] Verify image paths match filenames
- [ ] Check image names (case-sensitive!)
- [ ] Try opening image URLs directly

### **Problem: Mobile Menu Doesn't Work**
- [ ] Check `main.js` is uploaded
- [ ] Clear browser cache
- [ ] Test in different mobile browser
