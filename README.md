## ⚙️ Customization

### **Change Colors:**
Edit `css/style.css`, find `:root` section:
```css
:root {
    --primary-color: #2c3e50;    /* Dark blue */
    --secondary-color: #3498db;   /* Light blue */
    --accent-color: #e74c3c;      /* Red */
    --gold-color: #f39c12;        /* Gold */
}
```

### **Update Dates:**
Edit `index.html`, find the dates section:
```html
<p class="date">August 15, 2025</p>
```

### **Change Text:**
All content is in HTML files - easy to edit!
